## 制作DLL
DLL本身是动态链接库的意思，但是我参考了一款叫《While True: Learn()》的游戏，里面可以将一些基本函数封装成dll，以供复用

并且这么描述本身还是蛮贴合场景的，我也就这么沿用了。

快捷键：
选中多个节点后按Ctrl + G自动创建，要求只要要有一个Input节点才可用


——让多选变得再次伟大！

## 增加框选功能
嗯，现在只能框选，除此之外一无是处。

两种框选模式，参考了CAD软件：
从落点向右的所有框选都是局部框选（只要碰到一点就算是选中）（绿色细实线+透明绿色背景）
从落点向左的拖动是完全框选（只有完全框选才算选中）（细蓝色虚线+透明蓝色背景）
